Luxe Bath Bombs Shopify Store

Luxe Bath Bombs Shopify Web Store Repo on Github [ https://github.com/Sean-Drew/BB-react-shopify-store ]

Luxe Bath Bombs Shopify Web Store deployed front end [ https://sean-drew.github.io/BB-react-shopify-store/ ]

This is a Shopify web store to sell a variety of bath bombs. You'll find a nice landing page, a few ways to navigate to the product pages and a functional cart / product checkout. Built with React, this web store connects to the Shopify API to handle product checkout and payment processing. This is a dev project and not a real store, so please do not make any actual purchases. There were some complexities to work out with a routing bug I had at the time of deployment resulting from the way Github pages hosts projects. I'm particularly proud of the research, trial & error and general debugging that ultimately wound up resolving the issue because it was 2 issues that were tied into each other which made it difficult to find where exactly the problem was and how to solve it.

TPSY - Beer Buddy (Client)

TPSY Client Repo on Github [ https://github.com/Sean-Drew/TPSY-client ]

TPSY deployed front end [ https://sean-drew.github.io/TPSY-client/ ]

In the past few years the craft beer scene has exploded. Whenever I go to a packie (liquor store for non-Mass folks) I'm always seeing new beers with sharp packaging and impressive labels drawing me over, but it's just not realistic to buy the whole store! Sometimes I feel like I can't keep track of the beers I've already tried, the ones I want to try, and whether I liked them or not. Enter TPSY. Designed on a jQuery/Express platform, TPSY is a web application for beer aficionados to keep track of the beers they've tried, and to keep notes on them.

Free Icon Invaders

Free Icon Invaders Game Repo on Github [ https://github.com/Sean-Drew/free-icon-invaders ]

Free Icon Invaders deployed front end [ https://sean-drew.github.io/free-icon-invaders ]

What an addictive way to pass some time - a browser based invader-defense type game. Welcome to 'Free Icon Invaders'. Some functionality of the game required use of various js Math features which I hadn't used before, so it was nice to see some exposure to how some complex calculations are used. Working on a game is always fun because you can play it along the way, and at the end you've got a game you can really sink some time into. I like to say that a project is never finished, you just stop working on it. Whether I'm done working on this project is yet to be seen, it's good as it is now but I can definitely envision it as a mobile game if I decide to go in that direction in the future. If you play Free Icon Invaders just make sure you set a timer, otherwise you might accidentally spend the whole day spinning, shooting and scoring.
